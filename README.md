# Cursor Homepage Clone

A pixel-perfect recreation of the [Cursor](https://cursor.com) homepage, built using only **HTML** and **CSS** for practice.

---

## Sections Recreated

| Section               | Description                                                            |
| --------------------- | ---------------------------------------------------------------------- |
| **Navigation**        | Sticky navbar with logo, menu links, and Sign in/Download buttons      |
| **Hero**              | Main headline with CTA button and product screenshot                   |
| **Clients**           | Trusted by section with 8 company logos (Stripe, OpenAI, NVIDIA, etc.) |
| **Feature Sections**  | 3 alternating text + image sections (Agent, Tab, Ecosystem)            |
| **Reviews**           | 6-card testimonial grid from industry leaders                          |
| **Features Grid**     | 3-column cards (Models, Codebase, Enterprise)                          |
| **Changelog**         | Version updates with timeline                                          |
| **Join Us CTA**       | Team section with image                                                |
| **Recent Highlights** | Blog/news cards                                                        |
| **Final CTA**         | "Try Cursor now" section                                               |
| **Footer**            | 5-column links + theme toggle                                          |

## Fonts & Colors

### Typography

- **Primary Font**: `CursorGothic` (custom font) with `monospace` fallback

### Color Palette

| Variable               | Hex       | Usage                     |
| ---------------------- | --------- | ------------------------- |
| `--primary-bg`         | `#14120b` | Main background           |
| `--secondary-bg`       | `#1b1913` | Cards & sections          |
| `--primary-text-color` | `#edecec` | Main text                 |
| `--hover-text-color`   | `#b2b1b1` | Secondary text & hovers   |
| `--cta-btn-bg`         | `#edecec` | CTA button background     |
| `--cta-btn-text`       | `#14120b` | CTA button text           |
| `--cta-btn-hover-bg`   | `#d7d6d5` | CTA button hover          |
| Accent                 | `#f54e00` | Learn more links (orange) |
| Muted                  | `#8e8e8e` | Subtitles & timestamps    |
